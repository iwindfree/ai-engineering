# SmartWatch Pro X1

## 제품 개요

SmartWatch Pro X1은 프리미엄 스마트워치로, 건강 모니터링과 피트니스 트래킹을 위한 최신 기술이 탑재되어 있습니다.

## 주요 사양

| 항목 | 사양 |
|------|------|
| 디스플레이 | 1.9인치 AMOLED (450x450) |
| 배터리 | 450mAh (일반 사용 7일) |
| 방수등급 | IP68 (수심 50m) |
| 센서 | 심박수, SpO2, GPS, 가속도계 |
| 연결 | Bluetooth 5.3, Wi-Fi |
| 운영체제 | WearOS 4.0 |

## 가격

- 기본 모델: 349,000원
- 프리미엄 모델 (티타늄 케이스): 549,000원

- 미드나이트 블랙

- SmartWatch Pro X1 본체
