# EarBuds Air 3

## 제품 개요

EarBuds Air 3는 액티브 노이즈 캔슬링(ANC)과 고해상도 오디오를 지원하는 무선 이어버드입니다.

## 주요 사양

| 항목 | 사양 |
|------|------|
| 드라이버 | 11mm 다이나믹 드라이버 |
| 주파수 응답 | 20Hz - 40kHz |
| ANC | 하이브리드 ANC (-35dB) |
| 배터리 | 이어버드 8시간, 케이스 포함 32시간 |
| 충전 | USB-C, 무선충전 지원 |
| 코덱 | AAC, LDAC, aptX Adaptive |

## 가격

- 정가: 199,000원
- 번들 할인 (SmartWatch 동시 구매): 169,000원

## 색상 옵션

- 펄 화이트
- 오닉스 블랙
- 라벤더 퍼플

## 특징

- 공간 오디오 지원 (헤드 트래킹)
- 멀티포인트 연결 (2대 동시)
- IPX4 생활 방수
- 터치 컨트롤 커스터마이징
