# 배송 정책

## 배송 방법 및 기간

### 일반 배송
- 배송비: 3,000원 (5만원 이상 구매 시 무료)
- 배송 기간: 결제 완료 후 2-3 영업일
- 배송사: CJ대한통운, 한진택배

### 퀵 배송 (수도권 한정)
- 배송비: 5,000원
- 배송 기간: 당일 오후 2시 이전 주문 시 당일 배송
- 가능 지역: 서울, 경기, 인천

### 새벽 배송
- 배송비: 4,000원 (7만원 이상 구매 시 무료)
- 배송 기간: 밤 11시 이전 주문 시 다음 날 오전 7시 전 도착
- 가능 지역: 서울 전역, 경기 일부

## 배송 추적

- 배송 시작 시 SMS 및 앱 푸시 알림 발송
- 마이페이지 > 주문내역에서 실시간 배송 상태 확인 가능
- 배송 완료 후 1시간 내 수령 확인 알림 발송

## 해외 배송

현재 해외 배송은 지원하지 않습니다. 추후 서비스 확대 예정입니다.
