# 일반 자주 묻는 질문 (FAQ)

## 회원가입 및 계정

### Q1: 회원가입은 어떻게 하나요?

**A:** 하늘여행사 회원가입 방법입니다:

1. **웹사이트**: www.skytravel.co.kr → 우측 상단 "회원가입" 클릭
2. **모바일 앱**: "하늘여행" 앱 다운로드 후 회원가입
3. **방문**: 전국 지사 방문 시 현장 가입 가능

**필요 정보**:
- 이름, 휴대폰 번호, 이메일
- 여권 영문명 (해외여행 시 필수)

**혜택**:
- 신규 가입 시 10,000 마일리지 적립
- 생일 쿠폰 50,000원 할인권 제공

---

### Q2: 비밀번호를 잊어버렸어요.

**A:** 비밀번호 재설정 방법:

1. 로그인 페이지 → "비밀번호 찾기" 클릭
2. 가입 시 등록한 이메일 또는 휴대폰 인증
3. 임시 비밀번호 발급 → 로그인 후 변경

**고객센터 문의**: 02-1234-5678 (본인 확인 후 초기화 가능)

---

### Q3: 회원 탈퇴는 어떻게 하나요?

**A:** 회원 탈퇴 절차:

1. 로그인 → 마이페이지 → 회원정보 → 회원탈퇴
2. 탈퇴 사유 선택 → 비밀번호 확인 → 탈퇴 완료

**유의사항**:
- 진행 중인 예약이 있으면 탈퇴 불가
- 적립 마일리지는 소멸되며 복구 불가
- 재가입 시 기존 정보 복구 불가

---

## 마일리지

### Q4: 마일리지는 어떻게 적립되나요?

**A:** 하늘마일리지 적립 기준:

| 결제 방법 | 적립률 | 예시 |
|-----------|--------|------|
| 신용카드 | 1% | 100만원 결제 시 10,000 마일리지 |
| 체크카드 | 0.5% | 100만원 결제 시 5,000 마일리지 |
| 계좌이체 | 1% | 100만원 결제 시 10,000 마일리지 |
| 네이버페이 | 1% | 별도 네이버포인트 추가 적립 |

**추가 적립**:
- 리뷰 작성: 500 마일리지
- 사진 리뷰: 1,000 마일리지
- 친구 추천: 5,000 마일리지 (양측 모두)

---

### Q5: 마일리지는 어떻게 사용하나요?

**A:** 마일리지 사용 방법:

1. **여행 상품 할인**: 1 마일리지 = 1원 (최대 결제금액의 50%)
2. **상품권 교환**: 10,000 마일리지 = 신세계상품권 10,000원
3. **기부**: 희망브리지 전국재해구호협회 기부

**사용 조건**:
- 최소 사용: 5,000 마일리지 이상
- 유효 기간: 적립일로부터 3년
- 현금 환급 불가

---

### Q6: 마일리지 유효기간은 언제까지인가요?

**A:** 마일리지 유효기간:

- **적립일로부터 3년**
- 소멸 예정 마일리지는 매월 1일 이메일/SMS로 안내
- 소멸된 마일리지는 복구 불가

**소멸 방지 팁**:
- 마이페이지에서 소멸 예정 마일리지 확인
- 소멸 전 소액 상품 예약에 사용

---

### Q9: 카카오톡으로 상담할 수 있나요?

**A:** 카카오톡 상담 안내:

1. 카카오톡 → 검색 → "@하늘여행사" → 채널 추가
2. 채팅창에서 문의 내용 입력
3. AI 챗봇 자동 응답 또는 상담원 연결

**상담원 연결 시간**: 평일 09:00-18:00, 토요일 09:00-13:00

**가능한 서비스**:
- 상품 문의
- 예약 조회
- 예약 변경/취소 접수
- 긴급 상황 신고

---

### Q10: 여행 중 긴급 상황이 발생하면?

**A:** 해외 긴급 상황 대응:

**24시간 긴급 연락처**: +82-10-9999-5678

**대응 가능 상황**:
1. 의료 응급: 현지 병원 안내, 통역 지원, 보험 청구 안내
2. 사건/사고: 현지 경찰 신고 지원, 대사관 연락
3. 여권 분실: 임시 여권 발급 안내
4. 항공편 지연/취소: 대체 항공편 확보
5. 자연재해: 대피 안내, 귀국편 조정

**긴급 상황 시 준비물**:
- 예약 확인서 (예약번호)
- 여권 사본 (휴대폰 저장)
- 현지 호텔 주소
- 여행자 보험 증권

---

### Q12: 현금영수증 발급은 어떻게 하나요?

**A:** 현금영수증 발급 방법:

1. **예약 시**: 결제 단계에서 현금영수증 발급 선택 → 휴대폰 번호 입력
2. **예약 후**: 고객센터 문의 (결제 후 7일 이내)

**발급 조건**:
- 계좌이체, 현금 결제 시만 가능
- 신용카드 결제는 카드 매출전표로 대체

**조회 방법**: 국세청 홈택스 → 현금영수증 조회

---

### Q13: 세금계산서 발급이 가능한가요?

**A:** 세금계산서 발급 안내:

**발급 대상**: 사업자 (사업자등록증 필수)

**발급 절차**:
1. 예약 완료 후 고객센터 또는 이메일로 요청
2. 필요 서류: 사업자등록증 사본, 담당자 연락처
3. 발급 기간: 요청 후 영업일 기준 2-3일

**주의사항**:
- 여행 출발 후에는 발급 불가
- 현금영수증과 중복 발급 불가
