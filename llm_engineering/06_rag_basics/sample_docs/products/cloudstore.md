# CloudStore Basic

## 개요

CloudStore Basic은 개인 사용자와 소규모 팀을 위한 클라우드 스토리지 솔루션입니다. 간편한 파일 저장, 공유, 동기화 기능을 제공합니다.

## 주요 기능

### 스토리지 용량
- 무료: 10GB
- 유료: 최대 100GB

### 파일 공유
- 링크 공유 기능
- 비밀번호 보호 옵션
- 만료일 설정 가능

### 동기화
- 데스크톱 클라이언트 제공 (Windows, Mac, Linux)
- 모바일 앱 지원 (iOS, Android)
- 실시간 파일 동기화

### 보안
- AES-256 암호화
- 2단계 인증 지원
- SSL/TLS 전송 보안

## 가격

- 무료 플랜: 10GB, 기본 기능
- 스타터 플랜: $5/월, 100GB, 우선 지원

## 제한사항

- 최대 파일 크기: 2GB
- API 호출 제한: 1,000 requests/day
- 공유 링크: 10개까지 동시 활성화

## 사용 대상

- 개인 사용자
- 프리랜서
- 소규모 팀 (5명 이하)
